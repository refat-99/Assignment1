##Generics in TypeScript

Generics offer to devleloper to create reusable, flexible, and type-safe functions without losing type information.
Instead of writing separate code for different data types,
Generics one function or class work with many types keeping strict type checking.

Without Generics, developers:
 use duplicate code
 lose type safety
 use any (unsafe)
Generics solve these problems.

TypeScript safety:
use any;
no autocomplete
no type checking
reduce runtime bugs

Generics Work :
<T> is a type parameter.
It acts like a placeholder for a type.
T can be string
T can be number
T can be an object
T can be an array
it's can function reuseale and comfortable for devleloper.
