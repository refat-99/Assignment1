##The four pillars of Object-Oriented Programming help developers build scalable, maintainable, and clean applications.
1. Inheritance
2. Polymorphism
3 Abstraction
4. Encapsulation


#Inheritance:
Inheritance allows one class to re-use the properties and methods of another class.
Benefits of inheriance:
  1.Reduces duplicate code
  2.Improves code reusability
  3.Creates a clean hierarchy
  4.Easier maintenanc

#Polymorphism
Polymorphism means one method and many behaviors.
that means one man many caracter.

#Benefits
    1.Makes code flexible
    2.Easier feature extension
    3.Reduces complex if/else
